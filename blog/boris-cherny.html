Personnage

Boris Cherny

Auteur de Programming TypeScript chez O'Reilly, Staff Engineer chez Meta. Expert reconnu en systemes de types, infrastructure frontend et outillage developpeur.

Parcours

Boris Cherny est un ingenieur logiciel base dans la Bay Area, specialise en infrastructure frontend, systemes de types et outillage developpeur. Il a travaille chez plusieurs entreprises tech majeures avant de rejoindre Meta en tant que Staff Engineer.

Diplome de UC Berkeley en informatique, il s'est rapidement distingue par sa maitrise approfondie de TypeScript et des systemes de types statiques.

Programming TypeScript (O'Reilly, 2019)

Son ouvrage Programming TypeScript, publie chez O'Reilly, est devenu une reference incontournable pour les developpeurs souhaitant maitriser le systeme de types de TypeScript en profondeur. Le livre couvre :

  • Les fondamentaux du systeme de types TypeScript
  • Les types avances : generiques, types conditionnels, mapped types
  • Les patterns de conception type-safe
  • L'integration avec les ecosystemes React, Node.js et les outils de build

L'ouvrage se distingue par son approche pragmatique : il explique non seulement le "comment" mais surtout le "pourquoi" derriere chaque fonctionnalite du langage.

Expertise technique

  • Systemes de types : analyse statique, type inference, type safety a l'echelle
  • Infrastructure frontend : tooling, build systems, developer experience
  • TypeScript : contributeur reconnu a l'ecosysteme, conference speaker
  • Architecture logicielle : patterns pour les applications a grande echelle

Chez Meta

En tant que Staff Engineer chez Meta, Boris travaille sur l'infrastructure frontend et les outils de developpement utilises par des milliers d'ingenieurs. Son expertise en systemes de types contribue a ameliorer la fiabilite et la maintenabilite du code a l'echelle de l'entreprise.
