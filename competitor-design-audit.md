# RECOMMANDATIONS DESIGN POUR NEOPULSION

## Direction artistique recommandee : "Premium Bleu-Blanc avec touche de modernite IA"

### 1. HERO SECTION
**Recommandation** : Gradient bleu-blanc anime + headline bold

- S'inspirer de **Mister IA** : gradient bleu sophistique (blanc -> bleu doux -> bleu profond) avec un canvas anime subtil en arriere-plan (particules, lignes de connexion IA)
- Headline bold et direct comme l'actuel "On vous aide a devenir visible, ou on le fait pour vous" -- c'est excellent, le garder
- Double CTA : "Audit SEO Gratuit" (primaire, plein) + "Decouvrir nos formations" (secondaire, outline)
- Stats inline sous le hero : "324+ entreprises | 230K+ personnes formees | 10+ ans"

### 2. PALETTE DE COULEURS
**Recommandation** : Blanc/bleu gradient -- but avec une accent chaude

```
Primary:        #1E40AF (bleu profond)
Primary-light:  #3B82F6 (bleu vif)
Primary-ultra:  #DBEAFE (bleu tres clair, fonds de section)
Secondary:      #0F172A (navy fonce, textes)
Accent:         #F97316 (orange chaud) ou #8B5CF6 (violet IA)
Background:     #FFFFFF -> #F0F7FF (gradient blanc-bleu glace)
Surface:        #F8FAFC (gris bleu tres clair pour cards)
```

- Le bleu est dominant dans le secteur (7/10 concurrents) -- c'est attendu et synonyme de confiance
- L'accent orange ou violet differencie de la masse (peu de concurrents l'utilisent)
- Le violet connote l'IA et l'innovation (voir Mister IA, AI Sisters)

### 3. NAVIGATION
**Recommandation** : 5-6 items max + CTA header permanent

```
[Logo] | Accompagnement | Formations | Ateliers | Ressources | A propos | [CTA: Audit Gratuit]
```

- Mega-menu pour "Accompagnement" (SEO, SEA, Content, IA) et "Formations" (SEO, IA, Digital)
- **CTA header sticky** "Audit Gratuit" toujours visible au scroll (pattern vu chez 8/10 concurrents)
- Mobile : hamburger menu avec sous-sections accordeon

### 4. PRESENTATION DES SERVICES
**Recommandation** : Combiner framework numerote + cards

- Section "Notre methode" en 3 etapes numerotees (inspirer de Primelis) :
  - 01 -- Diagnostiquer (audit SEO/IA gratuit)
  - 02 -- Strategiser (plan d'action sur mesure)
  - 03 -- Executer & Former (accompagnement + montee en competence)
- Sous chaque etape, cards services avec icone, titre, description courte, fleche hover animee
- **Differenciateur Neopulsion** : montrer visuellement le lien entre accompagnement ET formation (double fleche, parcours progressif)

### 5. SOCIAL PROOF
**Recommandation** : 3 niveaux de preuve sociale

**Niveau 1 -- Logos clients** (juste sous le hero)
- Carousel automatique 15-20 logos, monochrome/gris au repos, couleur au hover
- "Ils nous font confiance" -- inspirer de Mister IA / AI Sisters

**Niveau 2 -- Stats impact** (section dediee)
- 3-4 grands chiffres animes au scroll (counter animation)
- "324+ entreprises accompagnees" | "230K+ personnes formees" | "10+ ans d'expertise" | "4.9/5 satisfaction"

**Niveau 3 -- Temoignages** (section dediee plus bas)
- Cards avec avatar, nom, poste, entreprise, etoiles, citation courte
- Layout grille 2-3 colonnes desktop
- **Ajouter** : un temoignage video si possible (differenciant fort, vu chez SavoirIA)

### 6. CTA STYLE
**Recommandation** : Gradient bleu + effet glow

```css
/* CTA Primaire */
background: linear-gradient(135deg, #1E40AF, #3B82F6);
color: white;
border-radius: 12px;
padding: 14px 32px;
box-shadow: 0 4px 20px rgba(59, 130, 246, 0.4);
transition: all 0.3s ease;

/* Hover : effet glow inspire de Deux.io */
box-shadow: 0 4px 30px rgba(59, 130, 246, 0.6);
transform: translateY(-2px);

/* CTA Secondaire */
background: transparent;
border: 2px solid #3B82F6;
color: #1E40AF;
```

- Textes CTA concrets : "Obtenir mon audit gratuit", "Voir les formations", "Reserver un appel"
- **Placement** : fin de chaque section (pattern unanime)

### 7. ELEMENTS DE DESIGN DIFFERENCIANTS A IMPLEMENTER

| Element | Inspiration | Impact |
|---------|-------------|--------|
| **Gradient anime en hero** | Mister IA | Perception premium immediate |
| **Mouse-tracking light effect sur cards** | Mister IA / Deux.io | Interactivite moderne |
| **Counter animation au scroll** | CyberCite / Deux.io | Stats plus memorables |
| **CTA glow pulse** | Deux.io | Attention CTA accrue |
| **Badge Qualiopi + OPCO** | AI Sisters / SavoirIA | Credibilite formation |
| **Theming couleur par section** | Mister IA | Distinction SEO vs Formation vs IA |
| **Logo carousel monochrome->couleur** | Eskimoz / AI Sisters | Social proof elegante |
| **Scroll fluide Lenis** | Mister IA / Deux.io | Experience premium |

### 8. STRUCTURE DE PAGE RECOMMANDEE

```
1. [NAVBAR STICKY]     - Logo + 5 items + CTA "Audit Gratuit"
2. [HERO]              - Gradient bleu-blanc + headline bold + double CTA + stats inline
3. [LOGOS CLIENTS]     - Carousel monochrome, auto-scroll
4. [METHODE 3 ETAPES]  - Framework numerote 01/02/03 avec cards
5. [ACCOMPAGNEMENT]    - Cards services SEO/SEA/Content/IA en grille
6. [FORMATION]         - Cards formations avec duree/niveau/prix + badge Qualiopi
7. [STATS IMPACT]      - 4 chiffres animes plein ecran, fond bleu fonce
8. [TEMOIGNAGES]       - Cards 3 colonnes avec etoiles
9. [BLOG/RESSOURCES]   - 3 derniers articles en cards
10. [CTA FINAL]        - Section gradient fonce "Pret a booster votre visibilite ?" + formulaire
11. [FOOTER]           - Liens, newsletter, reseaux, mentions legales
```

### 9. TYPOGRAPHIE RECOMMANDEE
- **Titres** : Inter ou Plus Jakarta Sans (moderne, geometrique, excellent rendu web)
- **Corps** : Inter ou DM Sans (lisibilite, pro)
- **Accent** : Optionnel -- une serif comme Lora pour les citations/temoignages (vu chez Mister IA)

### 10. CE QUI DIFFERENCIERA NEOPULSION

Le positionnement "Accompagnement + Formation" est rare et puissant. Aucun des 10 concurrents ne le fait aussi clairement. Voici comment le rendre visuellement evident :

1. **Double parcours visuel** en hero ou section dediee : "On le fait pour vous" (fleche gauche) <-> "On vous apprend a le faire" (fleche droite), avec un centre qui les connecte
2. **Badge "Eligible OPCO / Qualiopi"** visible des le hero (peu de concurrents le font aussi tot)
3. **Section "Pourquoi les deux ?"** avec comparatif visuel montrant la complementarite accompagnement + formation
4. **Palette violet accent pour les sections IA** -- positionne Neopulsion dans l'IA sans abandonner le bleu SEO
