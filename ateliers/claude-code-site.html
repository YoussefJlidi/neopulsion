Pourquoi cet atelier

Faire creer un site web coute entre 2 000 et 10 000 euros. Utiliser un page builder demande des heures d'apprentissage et produit souvent un resultat generique. Claude Code change la donne : vous decrivez ce que vous voulez en francais, et l'IA genere le site complet. Pas de drag-and-drop, pas de templates limites, pas de code a ecrire.

Cet atelier de 3 heures vous emmene de zero a un site en ligne. Vous n'avez besoin d'aucune competence technique. On commence par une demonstration en direct, puis chaque participant construit son propre site et le deploie sur internet avant la fin de la session. Ce n'est pas une maquette : c'est un vrai site, accessible publiquement, avec votre nom de domaine si vous en avez un.

Deroulement

0-30 min

Installation et prise en main

Installer Claude Code, comprendre l'interface et les commandes de base.

  • Installer Claude Code sur votre machine en suivant la procedure guidee pas a pas
  • Decouvrir l'interface en ligne de commande et les principes du dialogue avec l'IA
  • Comprendre la logique de fonctionnement : vous decrivez, l'IA code, vous validez
  • Configurer votre premier projet et preparer le brief de votre site
30-75 min

Demo live : site complet en 5min

Construction en direct d'un site web complet a partir d'un simple brief en langage naturel.

  • Observer la creation d'un site professionnel de A a Z en temps reel, a partir d'un brief de 3 phrases
  • Voir comment l'IA genere la structure des pages, le design, les textes et les interactions
  • Comprendre les techniques de briefing qui produisent les meilleurs resultats du premier coup
  • Poser vos questions en direct et voir l'impact de chaque instruction sur le rendu final
75-165 min

Construire et iterer son propre site

Chaque participant cree son site. Iterations, ajustements, personnalisation avec accompagnement.

  • Rediger le brief initial de votre site : activite, cible, pages souhaitees, ton editorial
  • Generer la premiere version et iterer en langage naturel pour affiner le design et le contenu
  • Ajouter des fonctionnalites specifiques : formulaire de contact, galerie, temoignages, carte
  • Beneficier d'un accompagnement individuel pour resoudre les points de blocage
165-180 min

Deploiement sur Netlify

Mise en ligne en un clic. Votre site est live avant de partir.

  • Creer un compte Netlify gratuit et connecter votre projet en quelques clics
  • Deployer votre site et verifier qu'il est accessible publiquement sur internet
  • Configurer un nom de domaine personnalise si vous en possedez un

Questions frequentes

Faut-il des connaissances en code pour participer ?
Absolument pas. C'est le principe meme de cet atelier : Claude Code genere tout le code a votre place. Vous decrivez ce que vous voulez en francais, l'IA produit le resultat. Aucune connaissance en HTML, CSS ou JavaScript n'est necessaire. Si vous savez ecrire un email, vous savez creer un site avec Claude Code.
Le site est-il vraiment en ligne a la fin de l'atelier ?
Oui. La derniere phase de l'atelier est consacree au deploiement sur Netlify. Votre site est accessible publiquement via une URL que vous pouvez partager immediatement. Si vous possedez un nom de domaine, on le connecte pendant la session. Sinon, Netlify fournit une URL gratuite en .netlify.app.
Peut-on modifier le site apres l'atelier ?
Oui. Vous repartez avec le code source complet de votre site et la methode pour le modifier avec Claude Code. Changer un texte, ajouter une page, modifier une image : il suffit de decrire la modification souhaitee en langage naturel et de redeployer. Les templates fournis vous guident pour les modifications les plus courantes.
